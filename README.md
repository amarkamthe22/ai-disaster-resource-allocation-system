🌍 AI Disaster Resource Allocation System

An intelligent web-based system that predicts disaster resource requirements using Machine Learning and provides smart recommendations for disaster management.

🚀 Features

- 🤖 AI-based disaster prediction (Shelters, Food, Rescue Teams, Casualties)
- 🚨 Risk level classification (Low, Medium, High)
- 💡 Smart AI recommendations for emergency response
- 📊 Dynamic data visualization using charts (Bar, Pie, Line)
- 📄 PDF report generation for predictions
- 🗺 Disaster simulation map for visualization
- 📍 Location detection (Geolocation API)

🛠 Technologies Used

- Backend: Python, Flask
- Machine Learning: Scikit-learn (Pickle Model)
- Frontend: HTML, CSS, Bootstrap
- Visualization: Chart.js
- PDF Generation: FPDF
- Other: JavaScript, Geolocation API

⚙️ How It Works

1. User enters disaster-related inputs (population, rainfall, damage, etc.)
2. Machine learning model predicts required resources
3. System calculates disaster risk level
4. AI provides recommendations for response
5. Data is visualized using interactive charts
6. User can download a detailed PDF report

🎯 Objective

To assist in efficient disaster management by predicting required resources and helping authorities take faster and smarter decisions.

🔮 Future Scope

- Integration with real-time disaster APIs
- Live weather and alert system
- User authentication system
- Database integration for history tracking
- Advanced analytics dashboard
