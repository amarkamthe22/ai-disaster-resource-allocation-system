About AI Disaster Resource Allocation System

The AI Disaster Resource Allocation System is a machine learning based web application designed to predict the resources required during disasters. This system analyzes disaster related parameters such as population, rainfall, damage level, and disaster type to estimate the required emergency resources.

The main goal of this system is to help authorities and disaster management teams make quick decisions about resource distribution during emergency situations.

🤖 Machine Learning

The system uses a Random Forest Machine Learning model to predict disaster resource requirements such as shelters, food packets, rescue teams, and casualties.

📊 Data Visualization

Interactive graphs and dashboards help visualize disaster data trends and resource requirements for better understanding and planning.

📄 Automated Reports

The system can generate disaster prediction reports in PDF format for documentation and analysis.

Project Features

  • AI based Disaster Resource Prediction
  • Risk Level Identification (High / Medium / Low)
  • Interactive Disaster Dashboard
  • Resource Visualization Graphs
  • PDF Disaster Report Generation
  • Dark Mode Support

Technologies Used

  • Python
  • Flask Web Framework
  • Machine Learning (Random Forest)
  • HTML, CSS, Bootstrap
  • Chart.js for Data Visualization
